## 完整頁面流程

### 入口 index.html
1. 邀請函畫面（QR Code 掃入）
2. 靈魂掃描（拍照）
3. 身分分配「你被分配到湯川秀樹的人生」
4. 劇情導入：識別證遺失
5. → 前往 exhibit1.html

---

### 展品頁共用結構（exhibit1~4）

每個展品頁的架構完全一致，只有內容不同：

| 區塊 | 說明 |
|------|------|
| ① 章節標題 | 展品編號、作品名稱 |
| ② 情境影片 | AI 生成影片（`<video>` 播放，檔案放 assets/videos/） |
| ③ 作品解說 | 湯川的人生場景說明 |
| ④ 問答選擇 | A / B 兩個選項，B = 湯川型（point +1） |
| ⑤ 撿到拼圖 | 回答後顯示故事情境文字 + 拼圖碎片動畫 |
| ⑥ 前往下一站 | 按鈕跳轉下一展品或結果頁 |

---

### 各展品內容對照

| 展品 | 作品 | 主題 | 問答情境 | 湯川型選項 |
|------|------|------|----------|------------|
| 1 | 〈一日生きる〉 | 過程導向 | 努力沒有回音時 | A：每天比昨天多做一點 |
| 2 | 〈見渡せば〉 | 存在叩問 | 深夜天橋上的車燈 | B：這些人這麼晚了，到底在忙什麼？ |
| 3 | 《世界は一つ》 | 整體性 | 分裂時代的直覺 | B：如果世界毀滅，真理毫無意義 |
| 4 | 《知魚樂》 | 直覺推導 | 混沌不明的難題 | A：想像應當存在的未知，再用數學證明 |

---

### 結果頁 result.html
1. 四片拼圖合體動畫
2. 融合照片（觀眾照 + 湯川濾鏡，目前為視覺呈現）
3. 靈魂匹配度揭曉（依 point 計算）
4. 介子理論哲學結語

---

## 計分與匹配度對照

每個展品選「湯川型」→ point +1，共 0–4 分

| point | 靈魂匹配度 | 結語調性 |
|-------|-----------|---------|
| 0 | 25% | 靈魂各異，卻因介子有所連結 |
| 1 | 43% | 部分共振，思想有所交會 |
| 2 | 61% | 明顯共鳴，你與湯川心有靈犀 |
| 3 | 79% | 高度同頻，靈魂已深度融合 |
| 4 | 97% | 近乎完全共振，你中有他，他亦有你 |

## sessionStorage 資料說明

| key | 說明 | 範例值 |
|-----|------|--------|
| `point` | 累積分數 | `"3"` |
| `playerPhoto` | 觀眾照片（base64）| `"data:image/jpeg;base64,..."` |
| `puzzle1` | 展品一拼圖是否獲得 | `"true"` |
| `puzzle2` | 展品二拼圖是否獲得 | `"true"` |
| `puzzle3` | 展品三拼圖是否獲得 | `"true"` |
| `puzzle4` | 展品四拼圖是否獲得 | `"true"` |

## 如何加入影片

將影片命名後放進 `assets/videos/` 資料夾：
```
assets/videos/exhibit1.mp4   ← 〈一日生きる〉
assets/videos/exhibit2.mp4   ← 〈見渡せば〉
assets/videos/exhibit3.mp4   ← 《世界は一つ》
assets/videos/exhibit4.mp4   ← 《知魚樂》
```
展品頁的 `<video>` 標籤已預留好路徑，放入影片後即可自動播放。
